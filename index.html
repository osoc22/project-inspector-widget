The project

Our application allows the economic inspectors at FOD economie to automatically check multiple websites at the same time and create automatic screenshots of the products that need to be checked. The application is able to create automatic reports which compare product prices during the period.

The feautures

When a user first visits the website, it is greeted with a log-in and sign-up page (See page above). Here the user can either register for a new account or login to their current account. After authentication, the user is given a quick overview of all its already created scrapers and their status.

On the same overview page, the user can add a new scraper to the list, which will extract the product data from the web page using the url entered by the user.
